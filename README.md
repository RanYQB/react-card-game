# Cards dealer


1. Clonez le projet :

```bash
  git clone https://github.com/RanYQB/react-card-game.git
```

2. Rendez-vous dans le répértoire du projet :

```bash
  cd react-card-game
```

3. Installez les dépendances:

```bash
  npm install
```

4. Lancez le serveur :

```bash
  npm run start
```
